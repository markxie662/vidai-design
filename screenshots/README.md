# VidAI Design 截图指南

## 📸 如何添加界面截图

### 方法 1: 使用截图辅助工具

运行以下命令启动截图辅助工具：

```bash
~/vidai/bin/vidai-screenshot
```

该工具会自动显示 VidAI 的各个界面，并在每个界面暂停，方便你截图。

### 方法 2: 手动截图

1. **启动 VidAI**
   ```bash
   vidai
   ```

2. **截取主界面**
   - 按 `Cmd + Shift + 4`
   - 选择界面区域
   - 保存为 `screenshots/main-interface.png`

3. **查看帮助界面**
   - 输入 `/help`
   - 按 `Cmd + Shift + 4` 截图
   - 保存为 `screenshots/help-interface.png`

4. **截取功能演示**
   - 尝试各种命令
   - 截取功能界面
   - 保存为 `screenshots/features.png`

### 推荐的截图文件名

- `main-interface.png` - 主界面截图
- `help-interface.png` - 帮助界面截图
- `features.png` - 功能展示截图
- `image-generation.png` - 图片生成过程
- `video-creation.png` - 视频创建过程
- `prompt-optimization.png` - Prompt 优化界面

### 截图要求

- **格式**: PNG（推荐）或 JPG
- **宽度**: 建议不超过 1200px
- **内容**: 清晰展示界面功能和美观度
- **风格**: 保持终端默认配色方案

### 自动添加到 README

截图添加到 `screenshots/` 目录后，会自动显示在项目 README 中。
